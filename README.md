# Sistem za E-Bolnicu / Opis projekta

Ovaj projekat je urađen u sklopu predmeta "**Metodologije razvoja softvera**" na Univerzitetu Singidunum.

Projekat obuhvata osnovne elemente jedne aplikacije za bolnicu, kao što su rad sa pacijentima, zaposlenima, terminima i dokumentacijom.

Model prikazuje kako bi izgledaо **prototip sistema** kroz dijagrame, dakle ovo nije gotova aplikacija.
