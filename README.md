# E-Bolnica

E-Bolnica je sistem namenjen radu bolnice i pacijenata.  
Obuhvata osnovne funkcionalnosti kao što su rad sa pacijentima i zaposlenima, zakazivanje termina i upravljanje medicinskom dokumentacijom.

U okviru projekta je prikazan prototip sistema kroz UML dijagrame i dizajn korisničkog interfejsa.  
Sistem nije implementiran kao gotova aplikacija.

## Ključne funkcionalnosti

Sistem obuhvata sledeće funkcionalne celine:

- Upravljanje pacijentima i medicinskim osobljem  
- Zakazivanje, pregled i otkazivanje termina  
- Rad sa medicinskom dokumentacijom  
- Razdvojeni portali za pacijente i lekare  
- Pregled svih zdravstvenih informacija na jednom mestu
  
## Sadržaj repozitorijuma

U okviru repozitorijuma nalaze se sledeći materijali:

- UML dijagrami:
  - Dijagram slučajeva korišćenja  
  - Dijagram klasa  
  - Dijagram arhitekture sistema
  - Sequence dijagram
  - Dijagram aktivnosti
  - Software Deployment dijagram 
- Dizajn korisničkog interfejsa (Figma prototipi)
